This plugin creates a shortcode for publishers within the advice network to create an impressions tracker in their posts without javascript knowledge.

== Installation ==

To install this plugin you only have to activate it. The shortcode will then be available in posts.

e.g.

1. Upload `advertorial.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[advertorial id=21312 site=site.com]` in your templates
